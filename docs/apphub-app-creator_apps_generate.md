## apphub-app-creator apps generate

Generate App Hub Applications

### Synopsis

Generate App Hub Applications based on CAIS Asset Search

```
apphub-app-creator apps generate [flags]
```

### Examples

```
Create apps by searching CAIS based on GCP Resource labels in the following locations: apphub-app-creator apps generate --parent projects/$project --management-project $mp --locations us-west1 --locations us-east1 --label-key $label_key

Create apps by searching CAIS based on GCP Resource tags in the following locations: apphub-app-creator apps generate --parent projects/$project --management-project $mp --locations us-west1 --label-key $tag_key --tag-value $tag_value

Create an application per Kubernetes namespace per GKE Cluster in the following locations: apphub-app-creator apps generate --parent projects/$project --management-project $mp --locations us-west1 --per-k8s-namespace=true

Create apps by searching Cloud Logging labels in the following locations: apphub-app-creator apps generate --parent folders/$folder --management-project $mp --locations us-west1 --log-label-key $log_label_key --log-label-value $log_label_value

Create one App Hub application per app.kubernetes.io/name label value: apphub-app-creator apps generate --parent projects/$project --management-project $mp --locations us-west1 --per-k8s-app-label=true

Generate a report of discovered assets: apphub-app-creator apps generate --parent projects/$project --management-project $mp --locations us-west1 --label-key $label_key --report-only=true

Automatically detect applications based on well known labels and tags: apphub-app-creator apps generate --parent projects/$project --management-project $mp --locations us-west1 --auto-detect=true --report-only=true

Generate an application per project or list of projects: apphub-app-creator apps generate --parent folders/$folder --management-project $mp --locations us-west1 --project-keys proj1 --project-keys proj2 --app-name my-app
```
